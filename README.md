# Detecting Malicious Twitter Bots Using Machine Learning

## Project Overview
Detecting Malicious Twitter Bots Using Machine Learning is a project that identifies automated or fake Twitter accounts (bots) and detects malicious URLs present in tweets. Social media platforms often suffer from spam, misinformation, and harmful links generated by bots. This project applies machine learning techniques to analyze tweet behavior and classify accounts as bot or genuine users.

The system uses the **Logistic Regression** algorithm to perform classification based on various tweet and account features.

---

## ✨ Objectives
- Detect malicious Twitter bot accounts.
- Identify tweets containing malicious URLs.

## 📁 Project Modules

### 1. Tweet Extraction
- Tweets are extracted from an offline Kaggle dataset.
- The system reads tweet data and prepares it for analysis.
- No internet connection is required since the dataset is stored locally.

### 2. Twitter Bot Detection using Machine Learning
The system analyzes three important features:

- **Activity:** Tweet frequency of the account.
- **Anonymity:** Account verification status and follower details.
- **Amplification:** Retweet count of tweets.

Based on these features, the model classifies accounts as **Bot** or **Normal User** using Logistic Regression.

### 3. Malicious URL Detection
- Tweets are analyzed to check the presence of URLs.
- Tweets containing excessive URLs are evaluated.
- The model predicts whether URLs are **Malicious** or **Non-Malicious**.

## 🎯 Model Performance
| Model | Accuracy |
|-------|----------|
| Bot Detection | ~74% |
| Malicious URL Detection | ~73% |
The performance of the model is evaluated using **ROC (Receiver Operating Characteristic) curves**, which compare the **True Positive Rate (TPR)** and **False Positive Rate (FPR)**.
